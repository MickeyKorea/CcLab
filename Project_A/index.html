Stop Covid Replication!

Generative Visuals with Interaction.

P5 Editor > Learn more >

This Project is midterm project for 2022 Spring Creative Coding Lab. User can kill the viruses with mouse click, and every time user kill ten viruses, new mutant viruses appear, and the mutant level increases correspondingly. Users must kill 100 viruses to finish the game, and if the viruses replicate over 60, the users will be failed.
